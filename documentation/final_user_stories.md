# Restaurant System User Stories (Final)

## EPIC: Table QR Ordering (Guest + Mobile)

- As a manager, I can generate and print a QR code for each table.
- As a guest, I can scan a table QR and open a web menu without installing an app.
- As a mobile user, I can scan the same table QR in the app and start ordering.
- Each QR scan carries the table token/number so all orders are linked to the correct table.
- While a table is occupied, guests can place multiple new orders and they are added to the same running table bill.
- A table stays occupied until the table bill is settled, or a manager reverses the table session.

---

## EPIC: Kitchen Operations

- As kitchen staff, I can use a Kitchen Display System (KDS) board that groups orders by status.
- As kitchen staff, I can view item notes/modifiers on the KDS card and prepare without verbal handoff.

---

## EPIC: Multi-Branch

- As an operator, I can manage branches and assign tables/staff/orders to a branch.
- As staff, I can filter KDS/operations per branch where needed.

---

## EPIC: Shift Management

- As a cashier, I can open and close shifts with opening/closing cash.
- As manager/admin, I can close cashier shifts and view reconciliation variance.

---

## EPIC: Menu Modifiers

- As a customer, I can select modifiers (size/spice/add-ons) while ordering.
- Modifier selections affect line-item price and are retained on order history/KDS.

---
